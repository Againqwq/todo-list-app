# \# 待办事项应用 (Todo List App)

# 

# 一个简单的命令行待办事项应用，帮助你管理日常任务。

# 

# \## 功能

# 

# \- ✅ 查看所有待办事项

# \- ✅ 添加新任务

# \- ✅ 标记任务为完成

# \- ✅ 删除任务

# 

# \## 安装和使用

# 

# \### 前置要求

# \- Python 3.6 或更高版本

# 

# \### 快速开始

# 

# 1\. 克隆仓库

# ```bash

# git clone https://github.com/Againqwq/todo-list-app.git

# cd todo-list-app

# ```

# 

# 2\. 运行应用

# ```bash

# python todo.py

# ```

# 

# 3\. 按照菜单提示操作

# 

# \### 示例

# 

# ```

# 🎯 待办事项应用

# 1\. 查看任务

# 2\. 添加任务

# 3\. 完成任务

# 4\. 删除任务

# 5\. 退出

# 请选择 (1-5): 2

# 输入任务: 完成 GitHub 项目

# ✅ 任务已添加: 完成 GitHub 项目

# ```

# 

# \## 文件说明

# 

# \- `todo.py` - 主程序文件

# \- `todos.json` - 存储待办事项的数据文件（自动生成）
